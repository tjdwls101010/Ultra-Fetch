## [0.2.5] - 2026-07-22

### Added
- Full onboarding documentation: a rewritten `README.md` (brand image, overview, features, quick
  start, usage, project status), a 9-page `docs/wiki` (Overview, Getting Started, Architecture,
  Concepts, CLI Reference, Troubleshooting & Reading Results, Design Decisions, Validation History,
  Development Guide), `LICENSE` (MIT), `CONTRIBUTING.md`, `CODE_OF_CONDUCT.md` (Contributor
  Covenant v2.1), and `SECURITY.md`.
- Repository description and topics set on GitHub for discoverability.

### Fixed
- `ultra_fetch.__version__` had been left at `0.1.0` since the project's first release and had not
  tracked the five releases since (`v0.2.0`–`v0.2.4`). Corrected to `0.2.5`.
